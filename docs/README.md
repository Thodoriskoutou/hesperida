# Hesperida Docs

Documentation site for Hesperida, built with Docusaurus (TypeScript).

## Install

```bash
npm install
```

## Local development

```bash
npm run start
```

## Build

```bash
npm run build
```

## Deploy

```bash
GIT_USER=<github-username> npm run deploy
```

This deploys the static build to GitHub Pages (`gh-pages` branch), using the settings from `docusaurus.config.ts`.
